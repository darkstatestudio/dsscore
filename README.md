# DssCore

Lightweight utility plugin for Unreal Engine 5 focused on reusable gameplay helpers,
common systems and performance-oriented tools.

# Features

## Utilities
- helper blueprint libraries
- gameplay utility functions
- math helpers
- async helpers
- debug helpers

## Gameplay Systems
- lightweight reusable components
- inventory/socket utilities
- pooling-friendly architecture
- subsystem helpers

## Performance Focused
- low overhead
- no unnecessary ticking
- subsystem-oriented design
- scalable architecture

---

# Supported Engine Versions

- Unreal Engine 5.6

# Setup

Enable plugin in:

```text
Edit -> Plugins
```
